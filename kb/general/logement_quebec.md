# Québec - Logement étudiant

## Description
Pour chercher un logement, l'étudiant peut regarder les résidences universitaires, les annonces hors campus et l'hébergement temporaire. Il doit lire attentivement le bail et vérifier les règles auprès des sources officielles en cas de doute.

## Source officielle
