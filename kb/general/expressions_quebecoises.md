# Québec - Expressions québécoises

## Description
Certaines expressions québécoises peuvent surprendre les nouveaux arrivants.

## Exemples
- "Bienvenue" peut signifier "de rien".
- "C'est correct" signifie souvent "ça va" ou "ce n'est pas grave".
- "À tantôt" signifie "à plus tard".
