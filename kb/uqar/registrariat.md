# UQAR - Registrariat et dossier étudiant

## Description
Le registrariat traite les demandes administratives liées au dossier étudiant, comme les attestations, les preuves d'inscription et les relevés de notes.

## Preuve d'inscription
Pour obtenir une preuve ou une attestation d'inscription, l'étudiant doit utiliser la procédure indiquée par l'UQAR pour les demandes d'attestation ou de relevé de notes. Le formulaire officiel mentionne l'attestation d'inscription et le dépôt au guichet étudiant ou l'envoi par courriel selon les coordonnées indiquées.

## Relevé de notes
Le même formulaire officiel permet de demander un relevé de notes. L'étudiant doit préciser le type de document demandé, le nombre de copies et son statut.

## Dossier étudiant
Pour une question précise sur le dossier étudiant, il faut contacter le registrariat, le guichet étudiant ou le campus concerné. NordikBot ne doit pas inventer de délai ou de frais si la base ne les contient pas.

## Questions fréquentes
### Comment obtenir une preuve d'inscription à l'UQAR ?
Utilisez la procédure de demande d'attestation ou de relevé de notes de l'UQAR. Si l'information manque ou si le cas est urgent, contactez le registrariat ou le guichet étudiant du campus.

### Comment obtenir un relevé de notes ?
Utilisez le formulaire officiel de demande d'attestation ou de relevé de notes, puis suivez les consignes de dépôt ou d'envoi.
